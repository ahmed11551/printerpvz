========================================
ФАЙЛЫ ДЛЯ УСТАНОВКИ РАСШИРЕНИЯ БРАУЗЕРА
========================================

Для установки расширения в Chrome/Edge нужно загрузить папку
со следующими файлами (через "Загрузить распакованное расширение"):

ОБЯЗАТЕЛЬНЫЕ ФАЙЛЫ:
-------------------
1. manifest.json          - манифест расширения (обязательно!)
2. content.js             - скрипт для перехвата ячеек
3. popup.html             - интерфейс popup
4. popup.js               - логика popup
5. background.js          - фоновый сервис

ОБЯЗАТЕЛЬНАЯ ПАПКА С ИКОНКАМИ:
-------------------------------
6. icons/icon16.png       - иконка 16x16
7. icons/icon48.png       - иконка 48x48
8. icons/icon128.png      - иконка 128x128

ИТОГО: 5 файлов + папка icons/ (3 файла) = 8 файлов

========================================
КАК УСТАНОВИТЬ:
========================================

1. Скопируйте ВСЕ эти файлы в одну папку
2. Откройте Chrome: chrome://extensions/
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами

========================================
ПРИМЕЧАНИЕ:
========================================

Расширение НЕ требует файлов сервера (server.py, config.json и т.д.)
Оно работает через HTTP с локальным сервером на localhost:5001
