# Размеры этикеток для Ozon и Wildberries

## Стандартные размеры

### Wildberries
- **Стандартный размер:** 58×40 мм ✅
- Также допускаются: 40×30 мм и 58×60 мм

### Ozon

**FBS (Fulfillment by Seller) - новый формат (с 14 мая 2024):**
- **Рекомендуемый размер:** 58×40 мм ✅

**FBO (Fulfillment by Ozon) - старый формат:**
- **Размер:** 75×120 мм

## Текущие настройки программы

Программа использует **58 мм** ширину этикетки, что соответствует:
- ✅ Стандарту Wildberries (58×40 мм)
- ✅ Новому формату Ozon FBS (58×40 мм)

### Где настраивается

В файле `config.json`:
```json
{
  "printer_settings": {
    "width_mm": 58,
    "label_height_mm": 40
  }
}
```

### Важно понимать

Для термопринтеров ESC/POS:
- **58 мм** - стандартная ширина термоленты/бумаги
- **80 мм** - более широкая термолента
- Ширина определяется **физической лентой** в принтере, а не настройками программы
- Программа просто использует соответствующую ширину печати

## Выбор размера принтера

Если у вас:
- **58 мм принтер** (например, Epson TM-T20, TM-T82) - используйте стандартные настройки
- **80 мм принтер** (например, Epson TM-T88) - можно использовать, но этикетка будет шире

## Рекомендации

Для работы с Ozon и Wildberries рекомендуется использовать **58 мм термопринтер**, так как:
1. ✅ Соответствует стандарту Wildberries
2. ✅ Соответствует новому формату Ozon FBS (самый распространенный)
3. ✅ Более компактные этикетки
4. ✅ Более экономичный расход бумаги
