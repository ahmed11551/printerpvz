# Инструкция по использованию EXE-версии

## Что это?

EXE-версия программы - это исполняемый файл, который **не требует установки Python** на компьютере пользователя.

## Что входит в комплект?

После создания EXE-файла через `СОЗДАТЬ_EXE.bat`, в папке `dist` будет:

- `Печать_Ячеек_ПВЗ.exe` - исполняемый файл сервера
- `ЗАПУСТИТЬ_EXE.bat` - скрипт для удобного запуска
- `config.json` - файл конфигурации принтера
- `README.md` - основная документация
- `INSTALL_RU.md` - подробная инструкция
- `ИНСТРУКЦИЯ_EXE.md` - эта инструкция

**ВАЖНО:** Расширение браузера (manifest.json, content.js, popup.js, background.js, icons) **не входит** в EXE-версию. Его нужно установить отдельно.

## Установка для пользователя

### Шаг 1: Распространение файлов

Скопируйте пользователю:
1. ВСЮ папку `dist` (или минимум файлы `Печать_Ячеек_ПВЗ.exe` и `config.json`)
2. Папку с расширением браузера (все файлы проекта: manifest.json, content.js, popup.js, background.js, icons/)

### Шаг 2: Настройка принтера

Пользователь должен открыть `config.json` и указать параметры своего принтера:

**Для Serial принтера:**
```json
{
  "type": "serial",
  "serial_port": "COM3",
  "serial_baudrate": 9600
}
```

**Для USB принтера:**
```json
{
  "type": "usb",
  "vendor_id": 1200,
  "product_id": 8258
}
```

**Для сетевого принтера:**
```json
{
  "type": "network",
  "network_host": "192.168.1.100",
  "network_port": 9100
}
```

### Шаг 3: Установка расширения браузера

1. Откройте Chrome: `chrome://extensions/` или Edge: `edge://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку с расширением (где находится manifest.json)

### Шаг 4: Запуск программы

**Вариант 1 (рекомендуется):** Двойной клик на `ЗАПУСТИТЬ_EXE.bat`

**Вариант 2:** Двойной клик на `Печать_Ячеек_ПВЗ.exe`

Сервер запустится на `http://localhost:5001`

**ВАЖНО:** Не закрывайте окно с программой! Сервер должен работать постоянно.

## Отличия от версии с Python

| Параметр | EXE-версия | Python-версия |
|----------|-----------|---------------|
| Требуется Python | ❌ Нет | ✅ Да |
| Размер файла | ~50-100 МБ | ~1 МБ |
| Установка зависимостей | ❌ Не нужна | ✅ Нужна |
| Обновление | Переустановка EXE | `pip install -r requirements.txt` |
| Настройка | config.json | config.json |

## Преимущества EXE-версии

✅ Не требует Python  
✅ Не требует установки зависимостей  
✅ Одна кнопка запуска  
✅ Подходит для неопытных пользователей  

## Недостатки EXE-версии

❌ Большой размер файла (50-100 МБ)  
❌ Может определяться антивирусом как подозрительный  
❌ Только для Windows  
❌ Для обновления нужен новый EXE-файл  

## Устранение проблем

### Антивирус блокирует EXE

Добавьте файл в исключения антивируса.

### EXE не запускается

1. Убедитесь, что `config.json` находится в той же папке, что и EXE
2. Проверьте права доступа (запустите от имени администратора)
3. Проверьте логи в папке `logs` (если она создалась)

### Принтер не работает

1. Проверьте настройки в `config.json`
2. Убедитесь, что порт принтера указан правильно
3. Проверьте подключение принтера к компьютеру

## Создание EXE (для разработчика)

1. Убедитесь, что установлен Python
2. Запустите `СОЗДАТЬ_EXE.bat`
3. Дождитесь завершения (несколько минут)
4. EXE-файл будет в папке `dist`
