========================================
УСТАНОВКА РАСШИРЕНИЯ БРАУЗЕРА
========================================

Эта папка содержит файлы для расширения браузера.

УСТАНОВКА В CHROME:
-------------------
1. Откройте Chrome
2. Введите в адресной строке: chrome://extensions/
3. Включите "Режим разработчика" (переключатель справа вверху)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите ЭТУ папку (где находится этот файл)

УСТАНОВКА В EDGE:
-----------------
1. Откройте Edge
2. Введите в адресной строке: edge://extensions/
3. Включите "Режим разработчика" (переключатель слева)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите ЭТУ папку

ВАЖНО:
------
- Сервер печати должен быть запущен на localhost:5001
- Расширение работает только на сайтах ozon.ru и wildberries.ru
